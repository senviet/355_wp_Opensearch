Add Open Search to your website has never been so easy.

== Description ==

OpenSearch is a collection of simple formats for the sharing of search results.

Simply, This plugin help you to make browser's addressbar become your search form.

Feature:

*   Admin Setting Page supported
*   Insert Searc metatab on your page's head.
*   Add rewrite for opensearch.xml

This plugin have a tutorial on Sen Viet : [VIẾT PLUGIN THÊM OPENSEARCH VÀO WORDPRESS](http://laptrinh.senviet.org/wordpress-plugin/viet-plugin-them-opensearch-vao-wordpress/)

== Installation ==

1. Upload wp-opensearc to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. To to plugin's setting page and done.
